# web

> 组态平台

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev | npm start

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
